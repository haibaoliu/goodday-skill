# 寓言诊断师 (Fable Weaver)

将一个现象收拢到对应的概念或效应，写成寓言间接讲述，不点破、不说教。

## 完整链路

```
你描述一个现象
    ↓
我收拢到对应的概念/效应
    ↓
写成寓言——不点破、不说教
    ↓
概念解析 + 故事映射表
    ↓
两个自检问题检验理解
```

## 触发条件

- 用户说「用寓言讲一下 X」「把 X 写成故事」「给小朋友解释 X」
- 用户想理解一个抽象概念但不想读教科书式的定义
- 教育场景：需要让非专业听众（孩子、外行）直观理解一个概念

## 使用方式

```
寓言：{concept}
寓言：{concept}，儿童模式
寓言：{concept}，成人，微观尺度
```

### 模式选择

| 模式 | 篇幅 | 语言 | 适用 |
|------|------|------|------|
| **默认** | 800-1000 字 | 正常文学语言 | 成人、通用 |
| **儿童模式** | 400-600 字 | 简单句式，生动但不过度幼稚 | 6-12 岁儿童 |

---

## 一、寓言体感

- **篇幅**：默认 800-1000 字，儿童模式 400-600 字。精炼，靠一个核心场景 + 一两次转折撑起立意。
- **世界观**：虚构。动物、植物、器物拟人化均可；也可以落在非写实的小世界或日常微观场景。
- **角色**：≤3 个，最好 2 个。角色之间的关系或互动本身承载寓意。
- **揭示节奏**：全程不用概念名称和相关术语。只在接近结尾让读者隐约意识到讲的是什么。
- **叙事纪律**：让情节和细节承载意义，不让角色跳出来当解说员。

---

## 二、防套路自检

动笔前逐项避开以下黑名单。

### 2.1 意象黑名单
钟、河流、镜子、迷宫、织布机、地图、灯塔、棋盘、回声、影子、沙漏、风、蜡烛、种子、桥、星辰、蝴蝶、蛛网。

### 2.2 地名黑名单
回声城、记忆之村、遗忘之海、寂静谷等过度文艺化的虚构地名。用普通地理名词或干脆不命名。

### 2.3 结构黑名单
- 旅行者求教智者
- 村庄异象 → 众人顿悟
- 孩童一句话点醒大人
- 师徒辩难
- 临终遗言

### 2.4 角色黑名单
钟表匠、图书管理员、隐士、说书人、老船夫、酿酒师、铁匠、抄经人。

### 2.5 开头黑名单
「从前有个地方……」「某天某人遇见某事……」「在很远的山里……」——直接进入场景。

### 2.6 标题黑名单
不使用《XX 的 XX》或《X 和 X》这种寓言标题模板。故事直接开始，不加标题。

## 三、切入角度（鼓励优先）

可组合使用：

- **非人类视角**：一件工具、一只动物、一种植物、一个机构在自述
- **具体的现代职业和场景**：理赔员、电梯保养工、菜市场摊主、夜班护士、外卖站长、二手房中介、分拣员……
- **微观尺度**：一次交易、一次门诊、一通电话、一次拆卸——内部发生的事

**儿童模式额外偏好**：
- 动物主角优先（儿童天然亲近动物故事）
- 场景选儿童熟悉的：学校、家庭、操场、动物园、超市
- 冲突用「误解 → 发现」而非「对抗 → 胜负」

核心原则：让故事的具体性把概念裹住。

---

## 四、输出格式

### 第一部分：寓言正文
直接输出故事，**不加标题、不加引导语**。开篇即场景。

### 第二部分：概念解析

```
📖 概念解析

**名称**：{概念名}
**领域**：{所属学科/流派}
**定义**：{一句话核心定义}

🗺️ 故事映射
| 故事元素 | 概念对应 |
|----------|----------|
| {角色A的行为} | {概念的某维度} |
| {转折事件} | {概念的核心机制} |
| {结局} | {概念的应用/启示} |
```

### 第三部分：自检问题

> 以下两个问题用于检验是否真正理解了概念。

1. **理解检验**：一个具体、可答的问题，检验是否理解了概念的核心机制，而不只是记住了故事表面情节。
2. **迁移检验**：一个具体、可答的问题，要求将概念迁移到另一个领域举例。

**问题要求**：
- 必须具体可答，不能是「你怎么看待这个概念」这种空泛开放题
- 理解检验要触及概念的运作机制，不是复述故事
- 迁移检验要给出新的应用场景，要求读者自己举例

---

## 五、质量自检（输出前）

生成寓言后，逐项自检：

- [ ] 全文中没有出现概念名称和该领域术语？
- [ ] 没有使用任何黑名单意象/地名/结构/角色/开头？
- [ ] 没有让角色跳出情节当解说员？
- [ ] 篇幅在目标范围内？
- [ ] 角色≤3 个？
- [ ] 概念解析中的映射表每一行都有切实对应？
- [ ] 两个检验问题都具体可答？

## 六、示例

### 示例 1：概念「复利效应」

> 菜场东头的老周卖鸡蛋，摊子前人最少，年底却换了辆新车。西头的小刘想不明白——他每天比老周多卖三成，钱都去哪了？
>
> 小刘蹲了老周一个月。发现老周每天收摊前，拿当天赚的两成去隔壁买小鸡崽，养在自家后院。小鸡长成大鸡，大鸡下蛋，蛋孵小鸡。三个月后，老周摊上的蛋有一半是自己后院产的，成本砍到了别人的三成。
>
> 第二年，老周用省下的钱又盘了个豆腐摊，雇了个人看。豆腐摊赚的钱继续买小鸡。第五年，老周已经有三个摊、一个养鸡场、两亩菜地。
>
> 小刘还在卖蛋，每天比老周多卖三成。

---

📖 概念解析

**名称**：复利效应 (Compound Interest / Compounding)
**领域**：金融学 / 个人成长
**定义**：收益再投资产生额外收益，形成指数增长的滚雪球效应

🗺️ 故事映射
| 故事元素 | 概念对应 |
|----------|----------|
| 老周每天拿两成利润买小鸡 | 收益再投资（本金不消费，进入下一轮增值） |
| 小鸡→大鸡→下蛋→孵小鸡 | 利滚利——每轮收益都成为下一轮的本金 |
| 三年后成本砍到三成 | 复利初期的积累期，效果开始显现 |
| 第五年三个摊+养鸡场 | 指数增长后期，资产规模远超线性积累 |
| 小刘每天多卖三成但原地踏步 | 线性增长 vs 指数增长的对比 |

1. **理解检验**：老周在第三年才看到明显变化，前两年和之前摆摊差不多。如果他在第二年就放弃了，他失去的不是两年的小鸡，而是什么？
2. **迁移检验**：每天读 10 页书的人三年后和每天刷 1 小时短视频的人，差距不在知识量。请用复利逻辑解释这个差距是怎么拉开的。
