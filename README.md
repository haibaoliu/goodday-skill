# Goodday Skills

A collection of practical [Hermes Agent](https://github.com/nousresearch/hermes-agent) skills for everyday productivity and development.

## Skills

### 📓 notebooklm
Interact with Google NotebookLM via CLI — login, list notebooks, query, and sync audio overviews.

### 🧠 agent-memory-systems
Knowledge base for evaluating, comparing, and choosing AI agent memory architectures (MemGPT, Mem0, Letta, etc.).

### 🐍 python-dependency-resolution
Resolve complex pip dependency conflicts — isolate, version-pin, and fix circular/impossible constraints.

## Usage

Copy any skill folder into your Hermes skills directory:

```bash
cp -r <skill-name>/ ~/.hermes/profiles/<profile>/skills/
```

Then Hermes will auto-discover and load them when relevant.
